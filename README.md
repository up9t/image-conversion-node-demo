# Image Convertion API with Nodejs Demo

This is a simple image conversion API with Nodejs. It uses the following libraries:

- [Express](https://expressjs.com/)
- [Sharp](https://sharp.pixelplumbing.com/)
- [Multer](https://github.com/expressjs/multer)
- [Pino](https://github.com/pinojs/pino)
- [Pino HTTP](https://github.com/pinojs/pino-http)

## Run with docker compose

1. Clone this repo:

    ```bash
    git clone --depth=1 https://github.com/up9t/image-conversion-node-demo.git
    ```

2. Run the following command:

    ```bash
    docker compose up -d
    ```

3. Go to http://localhost in your browser.
